**Gameplay**
------------

* Use the arrow keys or WASD keys to control your character's movement.
* Avoid & shoot enemies to stay alive.

**Technical Details**
--------------------

* **Game Engine**: Built using Godot 4.2.2 
* **Programming Language**: GDScript
* **Platforms**: Windows, Web (HTML5)
